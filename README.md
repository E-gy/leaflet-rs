# leaflet-rs

A [wasm-bindgen](https://github.com/rustwasm/wasm-bindgen)
wrapper for
[Leaflet.js](https://leafletjs.com/)

## Usage

```toml
[dependencies]
leaflet = "*"

[patch.crates-io.leaflet]
git = "https://github.com/kartevonmorgen/leaflet-rs"
```
